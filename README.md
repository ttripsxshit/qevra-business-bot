# QEVRA Business Bot

Commercial Telegram bot for business automation built with **Python** and **Aiogram 3**.

---

## Overview

QEVRA Business Bot is a production-ready Telegram bot template designed for service businesses.
It includes customer interaction, request processing, calculators and conversational workflows.

---

## Features

* Telegram Business Bot
* Service Calculator
* Project Request Form
* Client Chat
* FSM-based Dialogs
* JSON Storage
* Modular Architecture

---

## Tech Stack

* Python 3
* Aiogram 3
* FSM
* python-dotenv


## Installation

```bash
pip install -r requirements.txt
```

Create `.env`

```env
BOT_TOKEN=your_bot_token
ADMIN_ID=your_telegram_id
```

Run

```bash
python main.py
```
